Documentation
Introduction

This extension implements the Model Context Protocol (MCP) for Redmine, allowing Large Language Models (LLMs) to create and update issues in your Redmine instance through a Python-based API.

API Endpoints
Endpoint Method Description
/api/llm/create_issue POST Creates a new issue from a LLM prompt
/api/llm/update_issue/{issue_id} POST Updates an existing issue using LLM
/api/llm/analyze_issue/{issue_id} POST Analyzes an issue using LLM
/api/redmine/issues GET Retrieves a list of Redmine issues
/api/redmine/issue/{issue_id} GET Retrieves a specific Redmine issue
Example Usage
import requests # Create a new issue response = requests.post( 'http://localhost:5000/api/llm/create_issue', json={ 'prompt': 'Create a bug report for a login page error where users receive 404 error after login attempt' } ) print(response.json()) # Update an existing issue response = requests.post( 'http://localhost:5000/api/llm/update_issue/123', json={ 'prompt': 'Change the priority to high and assign to John' } ) print(response.json())
